# Seak Backend

Einfache Test-API für das Seak Frontend.

## Start

```bash
cd backend
pnpm install
pnpm run start:dev
```

API läuft auf http://localhost:3000.

## Endpoints

- `GET /api/arbeitszeiten` - Arbeitszeiten
- `GET /api/persons` - Personen
- `GET /api/planzeiten` - Planzeiten

## Tests

```bash
pnpm run test
```
